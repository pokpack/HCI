Human-Computer Interaction (HCI)

ภาคการศึกษา 69-1  ·  22 มิถุนายน – 12 ตุลาคม 2569  ·  เรียนทุกวันจันทร์ คาบละ 3 ชั่วโมง

15 คาบเรียน
8 บทเรียน
80% Hands-on
2 Module
Philosophy
ระดับ 1

Engineering Mindset

HCI คือวิศวกรรมที่ต้องวัดผลได้ ความสวยงามเป็นผลพลอยได้ ไม่ใช่เป้าหมาย

ระดับ 2

Evidence-Based Iteration

ทุก Prototype ต้องผ่าน Usability Testing → Evaluation → Metrics → Report ก่อนถือว่าสมบูรณ์

ระดับ 3

Real System Building

สร้างระบบจริง ทดสอบกับผู้ใช้จริง ทุกบทมี Deliverable ที่จับต้องได้

Learning Flow
ศึกษาผู้ใช้ → ออกแบบ → พัฒนาต้นแบบ → ทดสอบ → Usability Report → ปรับปรุง

วงจรนี้วนซ้ำทุกบท — Build → Test → Measure → Report → Refine

โครงสร้างรายวิชา
Module 1

GUI + Voice (บทที่ 1–4)

  • Ch.1  พื้นฐาน HCI และ Usability Engineering
  • Ch.2  UX Research และการวิเคราะห์ผู้ใช้
  • Ch.3  การออกแบบ UI และสร้างต้นแบบ
  • Ch.4  การปฏิสัมพันธ์ด้วยเสียง (Voice Recognition)
Module 2

Gesture + AR/VR (บทที่ 5–8)

  • Ch.5  การปฏิสัมพันธ์ด้วยท่าทาง (Gesture Recognition)
  • Ch.6  เทคโนโลยีความเป็นจริงเสริม (AR)
  • Ch.7  เทคโนโลยีโลกเสมือน (VR)
  • Ch.8  การบูรณาการเทคโนโลยีและโครงงานปลายภาค
