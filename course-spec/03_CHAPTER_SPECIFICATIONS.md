- จัดทำ Usability Report ได้

## Core Concepts
- Human-Computer Interaction
- User-Centered Design
- Usability Engineering
- ISO 9241-11
- Usability Metrics
- System Usability Scale (SUS)
- Usability Report

## Hands-on Labs
- วิเคราะห์ระบบจริง
- Mini Usability Testing

## Prototype
- Existing System Evaluation

## Deliverables
- Worksheet
- Mini Usability Report

## Assessment
- Worksheet
- Participation
- Usability Testing
- Report

- นักออกแบบคิดแทนผู้ใช้ได้
